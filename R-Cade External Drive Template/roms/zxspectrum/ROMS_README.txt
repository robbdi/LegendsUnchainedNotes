## SYSTEM ZXSPECTRUM ##
- - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - 
ROM files extensions accepted: .tzx  .tap  .z80  .rzx  .scl  .trd  .zip  .7z  .uce

Put your ZXSpectrum games in this directory that match the above extensions.
Some systems and/or games require BIOS to run properly. BIOS files should be placed in:
   /rcade/share/bios
