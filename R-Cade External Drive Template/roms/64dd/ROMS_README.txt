## SYSTEM NINTENDO 64DD ##
- - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - 
ROM files extensions accepted: .n64 .N64 .z64 .Z64 .v64 .V64 .bin .BIN .u1 .U1 .ndd .NDD

Put your Nintendo 64DD games in this directory that match the above extensions.

This system requires the following BIOS file to exist in order to run games:
   /rcade/share/bios/64DD_IPL.bin
