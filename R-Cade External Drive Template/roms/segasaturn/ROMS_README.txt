## SYSTEM SEGA SATURN ##
- - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - 
ROM files extensions accepted: .cue  .toc  .m3u  .ccd  .chd  .iso  .mds  .zip  .7z  .uce

Put your Sega Saturn games in this directory that match the above extensions.
Some systems and/or games require BIOS to run properly. BIOS files should be placed in:
   /rcade/share/bios
