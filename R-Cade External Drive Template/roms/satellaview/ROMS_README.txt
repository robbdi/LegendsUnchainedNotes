## SYSTEM SATELLAVIEW ##
- - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - 
ROM files extensions accepted: .bs  .smc  .sfc  .zip  .7z  .uce

Put your Satellaview games in this directory that match the above extensions.
Some systems and/or games require BIOS to run properly. BIOS files should be placed in:
   /rcade/share/bios
