## SYSTEM QUAKE ##
-------------------------------------------------------------------------------
ROM files extensions accepted: ".pak"

Put your Quake 1 .PAK games in this directory in numerical order.
PAK0.PAK must exist - and the included file should work for both shareware and full version.
The full version of the game requires the licensed PAK1.PAK to be added.
Additional paks should be added as the next number, for example: PAK2, PAK3, PAK4...
