## SYSTEM PANASONIC 3DO ##
- - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - 
ROM files extensions accepted: .iso  .cue  .chd  .zip  .7z  .uce

Put your Panasonic 3DO games in this directory that match the above extensions.
Some systems and/or games require BIOS to run properly. BIOS files should be placed in:
   /rcade/share/bios
