## SYSTEM SUPER NINTENDO ENTERTAINMENT SYSTEM ##
- - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - 
ROM files extensions accepted: .smc  .sfc  .mgd  .fig  .zip  .7z  .uce

Put your Super Nintendo Entertainment System games in this directory that match the above extensions.
Some systems and/or games require BIOS to run properly. BIOS files should be placed in:
   /rcade/share/bios
