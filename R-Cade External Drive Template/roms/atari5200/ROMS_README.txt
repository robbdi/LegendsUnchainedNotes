## SYSTEM ATARI 5200 ##
- - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - 
ROM files extensions accepted: .a52  .atr  .bas  .bin  .car  .dcm  .xex  .xfd  .atr.gz  .xfd.gz  .zip  .7z  .uce

Put your Atari 5200 games in this directory that match the above extensions.
Some systems and/or games require BIOS to run properly. BIOS files should be placed in:
   /rcade/share/bios
