## SYSTEM MGT SAM COUPé ##
- - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - 
ROM files extensions accepted: .dsk  .mgt  .sbt  .sad  .zip  .7z  .uce

Put your MGT SAM Coupé games in this directory that match the above extensions.
Some systems and/or games require BIOS to run properly. BIOS files should be placed in:
   /rcade/share/bios
