## SYSTEM SUFAMI TURBO ##
- - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - 
ROM files extensions accepted: .st  .smc  .sfc  .fig  .zip  .7z  .uce

Put your SuFami Turbo games in this directory that match the above extensions.
Some systems and/or games require BIOS to run properly. BIOS files should be placed in:
   /rcade/share/bios
