## SYSTEM NEO-GEO CD ##
- - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - 
ROM files extensions accepted: .cue  .zip  .7z  .uce

Put your Neo-Geo CD games in this directory that match the above extensions.
Some systems and/or games require BIOS to run properly. BIOS files should be placed in:
   /rcade/share/bios
