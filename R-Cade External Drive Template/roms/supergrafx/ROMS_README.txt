## SYSTEM SUPERGRAFX ##
- - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - 
ROM files extensions accepted: .pce  .sgx  .cue  .ccd  .chd  .zip  .7z  .uce

Put your Supergrafx games in this directory that match the above extensions.
Some systems and/or games require BIOS to run properly. BIOS files should be placed in:
   /rcade/share/bios
