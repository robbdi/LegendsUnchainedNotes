## SYSTEM AMSTRADCPC ##
- - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - 
ROM files extensions accepted: .dsk  .sna  .tap  .cdt  .voc  .m3u  .kcr  .zip  .7z  .uce

Put your AmstradCPC games in this directory that match the above extensions.
Some systems and/or games require BIOS to run properly. BIOS files should be placed in:
   /rcade/share/bios
