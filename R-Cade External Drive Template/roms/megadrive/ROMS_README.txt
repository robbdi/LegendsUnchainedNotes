## SYSTEM SEGA MEGADRIVE ##
- - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - 
ROM files extensions accepted: .md  .bin  .gen  .smd  .zip  .7z  .uce

Put your Sega Megadrive games in this directory that match the above extensions.
Some systems and/or games require BIOS to run properly. BIOS files should be placed in:
   /rcade/share/bios
