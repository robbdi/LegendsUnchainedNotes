## SYSTEM ATARI 2600 ##
- - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - 
ROM files extensions accepted: .a26  .bin  .zip  .7z  .uce

Put your Atari 2600 games in this directory that match the above extensions.
Some systems and/or games require BIOS to run properly. BIOS files should be placed in:
   /rcade/share/bios
