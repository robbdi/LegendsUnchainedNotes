## SYSTEM GAME BOY ##
- - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - 
ROM files extensions accepted: .gb  .zip  .7z  .uce

Put your Game Boy games in this directory that match the above extensions.
Some systems and/or games require BIOS to run properly. BIOS files should be placed in:
   /rcade/share/bios
