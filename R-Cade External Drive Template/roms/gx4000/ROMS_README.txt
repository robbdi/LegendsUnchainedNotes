## SYSTEM AMSTRAD GX4000 ##
- - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - 
ROM files extensions accepted: .cpr  .m3u  .zip  .7z  .uce

Put your Amstrad GX4000 games in this directory that match the above extensions.
Some systems and/or games require BIOS to run properly. BIOS files should be placed in:
   /rcade/share/bios
