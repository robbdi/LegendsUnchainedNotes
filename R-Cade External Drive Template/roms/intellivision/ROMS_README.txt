## SYSTEM MATTEL INTELLIVISION ##
- - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - 
ROM files extensions accepted: .int  .rom  .bin  .zip  .7z  .uce

Put your Mattel Intellivision games in this directory that match the above extensions.
Some systems and/or games require BIOS to run properly. BIOS files should be placed in:
   /rcade/share/bios
