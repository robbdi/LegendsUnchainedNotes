## SYSTEM WONDERSWAN ##
- - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - 
ROM files extensions accepted: .ws  .wsc  .pc2  .zip  .7z  .uce

Put your WonderSwan games in this directory that match the above extensions.
Some systems and/or games require BIOS to run properly. BIOS files should be placed in:
   /rcade/share/bios
