## SYSTEM SHARP X1 ##
- - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - 
ROM files extensions accepted: .dx1  .2d  .2hd  .tfd  .d88  .88d  .hdm  .xdf  .dup  .cmd  .zip  .7z  .uce

Put your Sharp X1 games in this directory that match the above extensions.
Some systems and/or games require BIOS to run properly. BIOS files should be placed in:
   /rcade/share/bios
