## SYSTEM THOMSON ##
- - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - 
ROM files extensions accepted: .fd  .sap  .k7  .m7  .rom  .zip  .7z  .uce

Put your Thomson games in this directory that match the above extensions.
Some systems and/or games require BIOS to run properly. BIOS files should be placed in:
   /rcade/share/bios
