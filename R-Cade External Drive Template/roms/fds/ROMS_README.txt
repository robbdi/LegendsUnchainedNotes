## SYSTEM FAMILY COMPUTER DISK SYSTEM ##
- - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - 
ROM files extensions accepted: .fds  .zip  .7z  .uce

Put your Family Computer Disk System games in this directory that match the above extensions.
Some systems and/or games require BIOS to run properly. BIOS files should be placed in:
   /rcade/share/bios
