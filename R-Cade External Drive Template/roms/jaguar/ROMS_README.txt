## SYSTEM ATARI JAGUAR ##
- - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - 
ROM files extensions accepted: .j64  .jag  .rom  .abs  .cof  .bin  .prg  .zip  .7z  .uce

Put your Atari Jaguar games in this directory that match the above extensions.
Some systems and/or games require BIOS to run properly. BIOS files should be placed in:
   /rcade/share/bios
