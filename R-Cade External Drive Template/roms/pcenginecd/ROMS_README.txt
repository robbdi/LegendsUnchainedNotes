## SYSTEM PC ENGINE CD ##
- - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - 
ROM files extensions accepted: .cue  .ccd  .chd  .zip  .7z  .uce

Put your PC Engine CD games in this directory that match the above extensions.
Some systems and/or games require BIOS to run properly. BIOS files should be placed in:
   /rcade/share/bios
