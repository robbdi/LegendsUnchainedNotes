## SYSTEM GAME BOY COLOR ##
- - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - 
ROM files extensions accepted: .gb  .gbc  .zip  .7z  .uce

Put your Game Boy Color games in this directory that match the above extensions.
Some systems and/or games require BIOS to run properly. BIOS files should be placed in:
   /rcade/share/bios
