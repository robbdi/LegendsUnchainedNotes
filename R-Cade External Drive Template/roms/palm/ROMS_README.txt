## SYSTEM PALM ##
- - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - 
ROM files extensions accepted: .prc  .pdb  .pqa  .img  .zip  .7z  .uce

Put your Palm games in this directory that match the above extensions.
Some systems and/or games require BIOS to run properly. BIOS files should be placed in:
   /rcade/share/bios
