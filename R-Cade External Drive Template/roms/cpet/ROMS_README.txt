## SYSTEM COMMODORE PET ##
- - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - 
ROM files extensions accepted: .d64  .d71  .d80  .d81  .d82  .g64  .g41  .x64  .t64  .tap  .prg  .p00  .crt  .bin  .gz  .d6z  .d7z  .d8z  .g6z  .g4z  .x6z  .cmd  .m3u  .vfl  .vsf  .nib  .nbz  .d2m  .d4m  .20  .40  .60  .a0  .b0  .rom  .zip  .7z  .uce

Put your Commodore PET games in this directory that match the above extensions.
Some systems and/or games require BIOS to run properly. BIOS files should be placed in:
   /rcade/share/bios
