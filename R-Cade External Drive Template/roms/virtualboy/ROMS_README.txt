## SYSTEM VIRTUAL BOY ##
- - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - 
ROM files extensions accepted: .vb  .vboy  .bin  .zip  .7z  .uce

Put your Virtual Boy games in this directory that match the above extensions.
Some systems and/or games require BIOS to run properly. BIOS files should be placed in:
   /rcade/share/bios
