## SYSTEM NINTENDO 64 ##
- - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - 
ROM files extensions accepted: .n64  .z64  .v64  .bin  .u1  .ndd  .zip  .7z  .uce

Put your Nintendo 64 games in this directory that match the above extensions.
Some systems and/or games require BIOS to run properly. BIOS files should be placed in:
   /rcade/share/bios
