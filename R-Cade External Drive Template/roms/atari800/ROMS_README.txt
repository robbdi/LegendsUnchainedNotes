## SYSTEM ATARI 800 ##
- - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - 
ROM files extensions accepted: .xfd  .atr  .atx  .cdm  .cas  .bin  .xex  .zip  .7z  .uce

Put your Atari 800 games in this directory that match the above extensions.
Some systems and/or games require BIOS to run properly. BIOS files should be placed in:
   /rcade/share/bios
