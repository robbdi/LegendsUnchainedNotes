## SYSTEM NINTENDO ENTERTAINMENT SYSTEM ##
- - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - 
ROM files extensions accepted: .nes  .dat  .zip  .7z  .uce

Put your Nintendo Entertainment System games in this directory that match the above extensions.
Some systems and/or games require BIOS to run properly. BIOS files should be placed in:
   /rcade/share/bios
