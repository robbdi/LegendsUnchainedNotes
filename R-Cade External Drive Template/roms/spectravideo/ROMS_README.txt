## SYSTEM SPECTRAVIDEO ##
- - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - 
ROM files extensions accepted: .cas  .bin  .zip  .7z  .uce

Put your Spectravideo games in this directory that match the above extensions.
Some systems and/or games require BIOS to run properly. BIOS files should be placed in:
   /rcade/share/bios
