## SYSTEM NEO-GEO POCKET COLOR ##
- - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - 
ROM files extensions accepted: .ngc  .ngp  .ngpc  .npc  .zip  .7z  .uce

Put your Neo-Geo Pocket Color games in this directory that match the above extensions.
Some systems and/or games require BIOS to run properly. BIOS files should be placed in:
   /rcade/share/bios
