## SYSTEM AMIGA 1200 ##
- - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - 
ROM files extensions accepted: .cue  .ccd  .iso  .hdf  .adf  .ipf  .lha  .dms  .dmz  .adz  .fdi  .nrg  .mds  .hdz  .lhz  .lzx  .m3u  .zip  .7z  .uce

Put your Amiga 1200 games in this directory that match the above extensions.
Some systems and/or games require BIOS to run properly. BIOS files should be placed in:
   /rcade/share/bios
