## SYSTEM MOVIES ##
- - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - - 
ROM files extensions accepted: .avi  .mp4  .ts  .flv  .mkv  .mov  .zip  .7z  .uce

Put your Movies games in this directory that match the above extensions.
Some systems and/or games require BIOS to run properly. BIOS files should be placed in:
   /rcade/share/bios
